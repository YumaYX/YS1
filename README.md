## Installation

```sh
gem build ys1.gemspec
gem install ys1*.gem --local
```

or

```sh
rake install:local
```

or

```sh
cat <<'GEMFILEEOF' >> Gemfile
gem 'ys1', git: 'https://github.com/YumaYX/YS1.git'
GEMFILEEOF
bundle install
```

## Usage

in your codes:

```ruby
require "ys1"

array = YS1::Csv.to_array("sample.csv")
puts array
```
